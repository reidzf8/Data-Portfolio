Tentang Saya

Saya mahasiswa Statistika dan Sains Data dengan kemampuan analisis data, interpretasi, dan visualisasi. Terbiasa mengolah data mentah menjadi insight yang disajikan melalui dashboard untuk mendukung pengambilan keputusan dan memberikan kontribusi nyata.

Keahlian Utama

Data Analysis

Eksplorasi, pembersihan, dan analisis data untuk menghasilkan insight yang relevan bagi pengambilan keputusan.

Statistical Modeling

Penerapan model statistik (regresi, ARIMA) untuk analisis hubungan dan peramalan data berbasis evidensi.

Data Visualization

Visualisasi data yang informatif dan komunikatif untuk menyederhanakan informasi kompleks.

Dashboard & Reporting

Pengembangan dashboard interaktif sebagai alat monitoring kinerja dan pendukung keputusan berbasis data.

Data Preprocessing

Transformasi data mentah menjadi data siap analisis melalui pembersihan, encoding, dan rekayasa fitur.

Infografis & Data Storytelling

Perancangan infografis berbasis data untuk menyampaikan insight secara ringkas, visual, dan mudah dipahami oleh audiens non-teknis.
